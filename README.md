# 📊 Data Cleaning Projects using Excel

This repository contains a collection of **real-world data cleaning projects performed using Microsoft Excel**.  
The focus is on transforming messy, inconsistent datasets into clean, structured, and analysis-ready data.

---

## 🚀 Objectives

- Practice real-world data cleaning techniques
- Improve Excel skills for data analysis
- Handle messy and unstructured datasets
- Prepare datasets for further analysis or visualization

---

## 🛠 Tools & Techniques Used

- Microsoft Excel
- Text to Columns
- Flash Fill
- Functions:
  - IF, IFS
  - TEXT, DATEVALUE
  - TRIM, SUBSTITUTE
  - VALUE
- Conditional Formatting
- Remove Duplicates
- Data Validation
- Sorting & Filtering

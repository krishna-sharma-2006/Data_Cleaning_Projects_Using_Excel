# 🎬 Data Cleaning Project: Movies Dataset (Excel)

This project demonstrates the process of cleaning a movies dataset using Microsoft Excel. 
The aim was to handle messy data, improve consistency, and prepare the dataset for analysis.

---

## 🚀 Objective

- Clean and structure movie dataset  
- Handle missing and inconsistent values  
- Prepare data for analysis  

---

## 🔍 Problems Identified

- Irregular row and column formatting  
- Line breaks within cells  
- Combined information in single columns (Director & Stars)  
- Missing values in Gross column  
- Extra spaces in text columns  
- Incorrect format in Year column  

---

## 🛠 Data Cleaning Steps

### 1. Fixed Row and Column Format
- Adjusted dataset structure  
- Ensured proper alignment of rows and columns  

---

### 2. Removed Line Breaks in Cells
- Cleaned cells containing unwanted line breaks  
- Made text consistent and readable  

---

### 3. Created Separate Columns
- Extracted Director and Stars into separate columns  
- Improved data usability  

---

### 4. Handled Missing Values in Gross Column
- Observed that ~95% values were missing  
- Since Gross is an important feature, values were filled with:
NA

---

### 5. Trimmed Stars Column
- Removed extra spaces using TRIM function  

---

### 6. Cleaned Year Column
- Removed unnecessary characters  
- Converted into proper numeric/date format  

---

## 🧰 Tools & Techniques Used

- Microsoft Excel  
- Functions: TRIM, LEFT, RIGHT, PROPER, REPLACE, FIND, VALUE, IF, IFERROR
- Techniques: Data cleaning, Column splitting, Missing value handling  

---

## 📈 Results

- Dataset is structured and consistent  
- Important columns preserved  
- Missing values handled appropriately  
- Data ready for analysis  

---

## 💡 Key Learnings

- Handling high percentage missing data  
- Importance of preserving critical columns  
- Data normalization techniques  
- Practical Excel data cleaning skills
